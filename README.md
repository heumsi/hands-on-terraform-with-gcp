# Hands on Terraform with GCP

Let's learn Terraform with GCP step by step!

## Development

### Requirements

- python 3.9.12

### Installation

```bash
$ python -m venv .venv
$ source .venv/bin/activate
$ (.venv) pip install -r requirements.txt
```
